Week 3 - Technical Blog

What is JavaScript and why is it so popular in web development?

JavaScript is similar to many computer-programming languages in the sense that it is very versatile in what it is able to do programmatically. For instance, we could write a function in JavaScript to multiply a number by two:

function double(x) {
return x * 2;
};

and we could invoke it like this:

double(5)
// returns 10

Now, we could do the same thing in Ruby, for example:

def double(x)
x * 2
end

and provoke it:

double(5)
# output: 10

So if JavaScript can do many of the same things other languages can, what makes it so popular in web development?

JavaScript is popular in web development because it is the only language that is supported on an internet browsers that can manipulate a webpage after that webpage is sent to a browser. JavaScript is sent with the webpage and can be useful for greater functionality once the page has been sent. For example, if we were to ask a user of a website to sign up with a username and password and also to verify their password, JavaScript would be the best choice to notify the user if their passwords match. Theoretically, we could ask the user to fill out their username, password, and password verification and hit the submit button to send their page to the server, and then finally have the server send back a result that would either tell the user they’ve failed or succeeded in creating their account…or we could have a nicer, less obtrusive solution which highlights the field green when the password matches and red when it doesn’t. Basically, JavaScript can make a webpage more interactive, and since it’s the only language that can support this client side dynamic, it is popular among web developers.
